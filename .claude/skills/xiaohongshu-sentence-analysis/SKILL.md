# 小红书英语句子分析 Skill

## 任务目标

接收英语原著句子、书名、作者名，生成可直接用于小红书发布的内容：

1. **5个推荐标题** — 格式：《书名》的"[核心特点]"，各有侧重
2. **正文分析** — 引言 + 修辞板块 + 词汇板块 + 句式板块，附emoji标记

## 触发场景

- 用户在 `http://localhost:8080/sentence-agent` 页面输入句子、书名、作者后请求生成小红书内容
- 用户直接请求分析某英语原著句子并生成小红书文案
- 任务涉及"小红书""标题""句子分析""英语原著解析"等

## 输入字段

| 字段 | 说明 | 示例 |
|------|------|------|
| `sentence` | 英语原著句子（完整原文） | "It was the best of times..." |
| `book` | 书名 | 双城记 |
| `author` | 作者名 | 狄更斯 |

## 执行方式

Claude Code 本身就是 LLM，直接按照下方格式生成内容即可，无需调用外部脚本。

如果用户环境有独立的 `Kimi_API_KEY`（标准 Moonshot key），可用脚本生成（见"脚本使用"章节）。大多数情况下直接生成更快、更可靠。

## 输出格式（严格遵循）

### 标题部分

生成5个推荐标题，每个从不同角度切入，格式固定：

```
推荐标题（5个）：
1. 《书名》的"[核心特点1]"
2. 《书名》的"[核心特点2]"
3. 《书名》的"[核心特点3]"
4. 《书名》的"[核心特点4]"
5. 《书名》的"[核心特点5]"
```

**角度覆盖建议**（5个标题各取其一，不重复）：
- 修辞手法（排比、对比、反讽、比喻等）
- 情感表达（张力、孤独、反叛等）
- 叙事技巧（悬念、留白、视角等）
- 哲理深度（人生观、命运、时代等）
- 语言特色（词汇、句式、节奏等）

**标题质量标准**：核心特点要新颖精准，2-5个汉字，让读者一眼看出这句话的独特价值。

### 正文部分

```
✨{author}《{book}》的[句子独特之处]！雅思[相关主题]的范本！

核心修辞：[修辞艺术主题]
✔️ [修辞手法名称]："[原文引用]"[效果分析，20字以内]
✔️ [修辞手法名称]："[原文引用]"[效果分析，20字以内]
✔️ [修辞手法名称]："[原文引用]"[效果分析，20字以内]

词汇宝库：[词汇主题]
💡 [词汇类别1]：
[英文词汇]（中文释义）
[英文词汇]（中文释义）
💡 [词汇类别2]：
[英文词汇]（中文释义）
[英文词汇]（中文释义）

句式突破：雅思高分表达模板
🌟 [句式类型]："[原文句式结构]"[应用说明]
🌟 [句式类型]："[原文句式结构]"[应用说明]
🌟 [句式类型]："[原文句式结构]"[应用说明]
```

## 各板块内容指引

### 核心修辞（3条）

分析句子中3个最突出的修辞手法，每条格式：
`✔️ 手法名称："原文引用"效果说明`

常见手法：对比（Antithesis）、排比（Parallelism）、比喻（Metaphor）、反讽（Irony）、夸张（Hyperbole）、拟人（Personification）、省略（Ellipsis）等。

### 词汇宝库（按类分组）

提炼句子中的高价值词汇，按词性或语义分类（高级形容词、动词短语、名词搭配、经典表达等），每词给出英中对照。

### 句式突破（3条模板）

提炼可直接用于雅思写作/口语的句型模板，每条格式：
`🌟 句式类型："[可复用的句式结构]"适用场景说明`

---

## 整体写作要求

- **纯文本**：不使用 Markdown 符号（**、#、-、>等）
- **字数控制**：全文不超过900字
- **语言风格**：既专业又有小红书亲和力，适合学英语的年轻人
- **雅思导向**：每个板块的分析都要点出雅思写作或口语的实用价值
- **引言吸睛**：第一句要突出这个句子自身最独特的亮点

## 示例

**输入：** sentence="It was the best of times, it was the worst of times" / book="双城记" / author="狄更斯"

**输出：**

```
推荐标题（5个）：
1. 《双城记》的"矛盾并置"
2. 《双城记》的"时代悖论"
3. 《双城记》的"排比震撼"
4. 《双城记》的"历史反讽"
5. 《双城记》的"开篇哲思"

✨狄更斯《双城记》的极致对比开篇！雅思写作排比句的绝佳范本！

核心修辞：对立统一的辩证美学
✔️ 对比（Antithesis）："the best...the worst" 极端对立词并列，震撼读者认知
✔️ 排比（Parallelism）：重复"it was the"句式，节奏铿锵，朗朗上口
✔️ 归纳（Generalization）：用"times"指代整个时代，视野宏大

词汇宝库：高级时代描述词汇
💡 形容词类：
the best（最好的） the worst（最坏的）
💡 名词类：
times（时代） age（时代） epoch（纪元）

句式突破：雅思高分表达模板
🌟 极端对比："It was the [最高级A], it was the [最高级B]" 描述矛盾现象首选
🌟 时代描述："It was an age of [抽象名词], it was an age of [对立名词]" 雅思开篇神句
🌟 排比递进："It was..., it was..., it was..." 三连排比增强论证气势
```

## 脚本使用（可选，需要独立 Kimi_API_KEY）

若环境中有 `Kimi_API_KEY`（`https://api.moonshot.cn/v1` 对应的密钥），可用脚本批量生成：

```bash
python .claude/skills/xiaohongshu-sentence-analysis/scripts/generate_xiaohongshu.py \
  --sentence "英语原著句子" \
  --book "书名" \
  --author "作者名"
```

脚本会从 `.env.local` 读取 `Kimi_API_KEY`，调用 Moonshot API 生成内容。
